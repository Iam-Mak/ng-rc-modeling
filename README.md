# Next Generation Reservoir Computing for ENSO Dynamics

This repository contains the code developed as part of my MSc thesis in Mathematical Modelling and Machine Learning.

The research investigates the application of **Next Generation Reservoir Computing (NGRC)** to the **El Niño–Southern Oscillation (ENSO)**, with the aim of understanding relationships between climate variables and exploring how data-driven models can capture and interpret complex climate dynamics.

## Overview

This work brings together concepts from:

- Machine Learning
- Reservoir Computing
- Nonlinear Dynamical Systems
- Climate Science

## Research Focus

- Reproducing and validating NGRC methodologies from existing literature
- Applying NGRC to ENSO related climate datasets
- Investigating nonlinear interactions between climate variables
- Analysing the influence of time delay structures
- Interpreting relationships learned by the model
- Comparing data-driven findings with established physical understanding of ENSO
